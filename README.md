# Sistema_de_votacao_Cepa
Sistema web de votação com registro de eleitores e candidatos, armazenamento em banco de dados MySQL e prevenção de votos duplicados.  Possui painel administrativo com autenticação, exibição de resultados em tabela e gráfico, atualização automática e edição segura de candidatos.  Projeto focado em organização e segurança no processo de votação.

Hospedagem: cepa.great-site.net
